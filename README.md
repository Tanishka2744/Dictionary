## Dictionary App

A simple Dictionary web application that allows users to search for word meanings
using a public dictionary API.

### Features

- Search word meanings instantly
- Displays definition, part of speech, and examples
- Clean and user-friendly interface
- Error handling for invalid or empty input

### Tech Stack

- HTML5
- CSS3
- JavaScript
- Dictionary API

### What I Learned

- Fetching data from APIs
- Handling user input and errors
- DOM manipulation using JavaScript
